# Development Environment as Code

This project provides a reusable devcontainer setup designed to make developers happy by simplifying and standardizing the development environment across projects. Think of it as "development environment as code"—similar to infrastructure as code or company as code, but focused on your daily developer experience.

## Purpose

- **Reusable Devcontainer**: Easily integrate this devcontainer configuration into any project to ensure a consistent, ready-to-code environment for all developers.
- **Developer Happiness**: Reduce onboarding time, eliminate "works on my machine" issues, and empower teams to focus on building features instead of troubleshooting setups.
- **Best Practices**: Encapsulate tools, extensions, and settings as code, making your development environment versioned, portable, and shareable.

## Getting Started

1. Use [VS Code Dev Containers](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) to open this folder or reuse its configuration in your own projects.
2. Enjoy a seamless, reproducible development experience.
